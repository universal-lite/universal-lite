Custom Component Documentation

Contributor and agent reference for Universal-Lite's custom UI surfaces. These docs are not end-user help; they exist to make ownership, source anchors, visual states, and safe-edit boundaries fast to inspect before changing component code.

Information-density rule Prefer dense visual structures over prose when they make ownership, flow, state, or safe-edit constraints faster to understand.
