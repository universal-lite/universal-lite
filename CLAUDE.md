# Universal-Lite CLAUDE.md

## Memory

**Do NOT manually write to `~/.claude/projects/.../memory/MEMORY.md`.**
It is auto-generated and will be overwritten. Use the MCP memory tool to record memories instead:
```
mcp__claude-memory__memory action=record project=ublue-mike
```

Use `mcp__claude-memory__recall` with tags to retrieve them (see MEMORY.md Quick Recall section for current tags).
